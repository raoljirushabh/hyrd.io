#**✦ Hyrd**

Hyrd is an AI-powered job application assistant that tailors your resume and cover letter to any job description — strategically, subtly, and in a way that reads like a human wrote it.
It targets maximum ATS compatibility while keeping your authentic voice intact. No robotic phrasing. No obvious AI tells. Just a sharp, well-positioned application, every time.

Features:

Resume Tailoring — rewrites your resume to mirror the JD's language, priorities, and keyword structure without sounding forced
Cover Letter Generation — tone-matched to the company (startup, corporate, creative, or academic) with a natural human voice
Application Q&A — answers custom application questions in your voice, grounded in your background and the role
ATS Score — real-time match confidence score with a breakdown across keyword coverage, skills alignment, experience relevance, and formatting
Keyword Integration — three-tier keyword placement strategy; every JD term woven in naturally, never stuffed
STAR Format Bullets — adds missing bullets in Situation/Task/Action/Result format, drawn from JD responsibilities, structured so the format is invisible
Grammar & Consistency Fixes — tense, punctuation, date formatting, and capitalization corrected silently throughout
PDF & DOCX Resume Upload — parse your existing resume directly from a file
Export Options — copy to clipboard, download as .txt, or download as a formatted .docx (proper font hierarchy, section headers, bullet styles)


Tech Stack: 

Layer --> Technology
UI Framework --> React 18
AI --> Anthropic Claude API (claude-sonnet-4)
PDF Parsing --> PDF.js
DOCX Parsing --> Mammoth.js
DOCX Generation --> docx.js
Styling --> Inline styles (no CSS framework dependency)
Deployment --> GitHub Pages / Vercel / Netlify
